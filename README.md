# REU 2026: Introduction to Data Science in R

This repository contains the lessons included in the 4-day *Introduction to Data Science in `R`* workshop delivered to the 2026 JHU Biology REU cohort. The goal of this workshop is to introduce and familiarize the basics of working with simple data structures in the `R` programming language. 

By the end of this workshop, students will be familiar with:
1. `R`, `RStudio`, and Posit Cloud
2. Creating and importing data into data frames
3. Manipulating data frames
4. Computing basic statistics and plotting data
5. Interpretting and reproducing common plotting formats used in biology

The lessons structure for the workshop is as follows:
1. **Introduction to R**
2. **Tidyverse**
3. **Probability and statistics**
4. **Biological data**
